# overwatch-chracters
> 게임 오버워치의 캐릭터들을 볼 수 있는 사이트입니다.

## 사용스킬
스킬 | 내용
:--: |--
parcel-bundler | 개발 도구
scss | css의 전처리도구로 css보다 더 간단하고 편리하게 사용할 수 있다.
variable of scss | scss의 변수를 이용하여 자주 사용하는 url주소르 저장
for of scss | scss의 반복문을 이용하여 32가지의 캐릭터 설정을 5줄로 코딩
